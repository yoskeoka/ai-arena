# Project Plan: AI Arena

## Goal

人間がプレイせず、人間が開発したAIが代わりにプレイするオンラインゲームを開発する。参加者は優秀なAIの開発を競い合い、観戦者はリアルタイムでゲーム画面を観戦できる。

## Significance

- AI開発の腕を競い合う新しい形の競技プラットフォーム
- LLMに丸投げできない高速判断が求められる設計により、本質的なAIアルゴリズム開発力が試される
- 観戦可能なゲーム画面により、AI同士の対戦をエンターテインメントとして楽しめる
- AIの開発・改善サイクルを通じて、参加者のプログラミングスキル向上にも寄与

## Requirements

### ゲーム形態
- **高速判断**: LLMに丸投げできないよう、1秒以下程度の高速な判断が求められる設計
  - ターン制を想定: 1秒に1ターン進行し、1秒以内に応答できなかった場合は行動スキップ扱い
  - リアルタイムではなくターン制とすることで、公平性と実装の明快さを担保
- 具体的なゲーム内容は未定（別途デザインフェーズで決定）

### AI参加ルール
- ゲームを直接プレイするAIはプレイ中にアップデート（自己改善）してもよい
- **外部通信による情報・命令の取得は禁止**（不正防止）
- **情報の外部送信（ログ・メトリクス送信）は許可**（デバッグ・分析用途）

### 観戦機能
- 観戦者に何が起きているかわかるように、実際に動いているゲーム画面をリアルタイム表示
- ゲームの状況が視覚的に理解しやすいUI

### プラットフォーム
- オンラインで複数のAIが参加・対戦できるインフラ
- AI接続用のAPIインターフェース
- マッチメイキングやリーグ/トーナメント形式の対戦管理

- [ ] Phase 2: ゲームエンジン（サーバーサイド）の実装
- [ ] Phase 3: AI接続APIの設計・実装

- [ ] Phase 5: マッチメイキング・対戦管理システム
- [ ] Phase 6: サンプルAIの作成・ドキュメント整備
- [ ] Phase 7: オンライン対戦インフラの構築・デプロイ
